Simple Unity 4.6+ setup:
- Add the FeedMeCanvas prefab to your scene
- Set the Feed URL to the url to your feed. (It requires a CrossDomainPolicy.xml on server)
- Set Type to the feed type or leave as auto.
- Set the time (in minutes) between feed updates or set to Update Once.
- Set the Feed Length to the number of feed items you wish to load or set to -1 to load all items.

Online documentation of classes and methods: https://dl.dropboxusercontent.com/u/43996324/Unity/FeedMe/Documentation.html
